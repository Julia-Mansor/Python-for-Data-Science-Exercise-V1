# Python for Data Science Exercise V1

## Overview

This repository contains a practice final examination for a **Master of Data Science** course. The exercise is designed to assess fundamental Python programming and data analysis skills using **Pandas** and **Matplotlib**.

The questions cover common data science tasks that are frequently used in academic assignments and real-world data analysis.

## Topics Covered

* Merging two DataFrames
* Calculating averages
* GroupBy operations
* Pivot Tables
* Writing Python functions
* Multi-Line Charts
* Box Plots
* Data binning using `pd.cut()`
* Pie Charts with percentage labels (1 decimal place)

## Learning Outcomes

By completing this exercise, students will be able to:

* Manipulate and combine datasets using Pandas.
* Perform basic descriptive statistics.
* Summarize data using GroupBy and Pivot Tables.
* Develop reusable Python functions.
* Visualize data using various chart types in Matplotlib.
* Categorize continuous data into bins.
* Create informative charts for data presentation.

## Technologies Used

* Python 3
* Pandas
* NumPy
* Matplotlib

This repository is intended for educational purposes and serves as a revision exercise for students studying Python for Data Science.
